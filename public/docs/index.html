Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://localhost

Authenticating requests

This API is not authenticated.

Product Management

Display a listing of Products.

Example request:
curl --request GET \
    --get "http://localhost/api/products/index/10" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/products/index/10"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

    method: "GET",
    headers,

Store a newly product in storage.

Example request:
curl --request POST \
    "http://localhost/api/products/store" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "name=dicta" \
    --form "description=sunt" \
    --form "image=@C:\Users\iteam\AppData\Local\Temp\php5ED7.tmp" 
const url = new URL(
    "http://localhost/api/products/store"
);

const headers = {
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('name', 'dicta');
body.append('description', 'sunt');
body.append('image', document.querySelector('input[name="image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());

Update user profile.

Must be authenticated

Example request:
curl --request PUT \
    "http://localhost/api/auth/me" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"uqv\",
    \"password\": \"pwswr\",
    \"gender\": \"female\",
    \"city\": \"natus\",
    \"bio\": \"ut\"
}"

const url = new URL(
    "http://localhost/api/auth/me"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "uqv",
    "password": "pwswr",
    "gender": "female",
    "city": "natus",
    "bio": "ut"
};

fetch(url, {
    method: "PUT",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
Received response:
Request failed with error:
